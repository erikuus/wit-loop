Eesti

Can AI write a witty sentence?

This is an experiment with one unfinished sentence: can AI complete it in a way that a person would recognize as wit?

Given beginningi’m lonely, please ...

The sentence comes from the American writer Melissa Broder.

The Criteria

To judge wit, the AI wrote its own criteria. They seem surprisingly sensible.

Reversal and logic

The ending should turn the opening in a way that is surprising but still clean.

Precision

It should feel as if no word can be replaced without making the line less exact.

Compression

The line should do a lot with very few words.

Quotable shape

The full sentence should sound finished and memorable.

Human truth

The joke should reveal something real about people, not just juggle words.

Double tone

The line should feel funny and sad at once, playful on the surface and deeper underneath.

Elegance

The line should move lightly rather than feeling clumsy or overbuilt.

The Loop

The loop is the heart of this experiment. How does it work, and why is it necessary? If we asked the same AI conversation to keep proposing new versions and also judge them, it would start remembering its own earlier thoughts, preferring paths it had already taken, and defending its own decisions. To avoid that, we split the work into three parts. The first two components, the creator and the judge, are AI agents that start at fixed intervals. When the judge is done, an ordinary script performs the final step. Then, after some time, the cycle repeats.

Creator

An AI agent that sees only the beginning of the sentence. It does not see the criteria, earlier attempts, or the current best line. It repeats the same task each time — complete the unfinished sentence so the result is witty — as if seeing it for the first time.

Judge

An AI agent that sees one complete sentence and the judging criteria. It does not see earlier attempts or earlier scores. Each time it has one simple task: judge one sentence by the criteria.

Checker

A script that runs after the judge. It recalculates the score, compares it with the current best one, and replaces the winner only if the number is higher.

What AI Thought Was Best

After many attempts, the loop considered this its best sentence:

i’m lonely, please stop calling my two-person table 'optimistic seating.'

The judge gave it 81/100.

The Original

I am lonely, please leave me alone.

It is almost perfect wit.

The turn is sharp and unexpected, the wording short and exact, almost mirror-built. The sentence stays in memory at once. The second half seems to mock the first, yet it also makes the line deeper than ordinary complaining, catching the contradiction in the human need to seek closeness and to pull away from it.
